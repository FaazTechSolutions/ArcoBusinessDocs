# Employee List Module
## Business Understanding Document

---

# 1. Module Overview

The Employee List module is used to manage all employees available in the HRM (Human Resource Management) system.

This module acts as the central employee master database.

All employee-related operations start from this screen.

---

# 2. Business Purpose

The Employee List module helps the organization to:

- Store employee master records
- Search employees quickly
- Track employee identification details
- Access employee contracts
- Manage employee lifecycle
- Connect employees with HR, payroll, lodging, and contract operations

---

# 3. Screen Overview

## Navigation Path

HRM → Employee → Employee List

---

# 4. Main Functions Available

| Function | Purpose |
|---|---|
| View Employees | Display all employee records |
| Search Employee | Search employee using employee details |
| Employee Selection | Select employee for actions |
| Pagination | Navigate through large employee records |
| Employee Details Access | Open employee profile information |
| HR Operations | Perform employee-related transactions |

---

# 5. Employee List Columns

The system displays employee information in tabular format.

| Column | Description |
|---|---|
| Employee ID | Unique system-generated employee number |
| Name | Employee full name |
| Iqama Number | Saudi residency identification number |
| Country Entrance No | Immigration or visa entry number |
| Candidate ID | Recruitment candidate reference number |
| KSA Arrival Date | Employee arrival date to Saudi Arabia |

---

# 6. Business Understanding of Each Field

## Employee ID

- Primary employee reference in system
- Used in all modules
- Unique for every employee

Example:
- 1001164
- 1001169

---

## Name

Employee full legal name.

Used for:
- HR records
- Contracts
- Payroll
- Legal documents

---

## Iqama Number

Saudi Arabia residency identification number.

Important for:
- Government compliance
- Visa tracking
- Contract validation
- Employee renewal process

---

## Country Entrance Number

Immigration entry reference number.

Used for:
- Travel tracking
- Arrival verification
- Visa processing

---

## Candidate ID

Recruitment process reference number.

Connected with:
- Recruitment module
- Candidate onboarding
- Agency processing

---

## KSA Arrival Date

Date employee entered Saudi Arabia.

Important for:
- Residency calculation
- Probation tracking
- Payroll eligibility
- Contract activation

---

# 7. Business Flow

## Employee Lifecycle Flow

Recruitment → Candidate → Arrival → Employee Creation → Contract Assignment → Payroll → Renewal/Retirement

---

# 8. Employee Creation Process

Employee records are usually created after:

1. Candidate recruitment completed
2. Visa approved
3. Employee arrived in KSA
4. HR onboarding completed

Then:
- Employee ID generated
- Employee profile created
- Employee becomes available for operations

---

# 9. Search Functionality

The search box allows users to search employees using:

- Employee ID
- Name
- Iqama Number
- Candidate ID

---

# 10. Employee Selection

Users can select employees using checkbox selection.

Selected employees can be used for:

- Contract assignment
- Batch operations
- Export
- Transfer
- HR actions

---

# 11. Action Menu

The "Action" button is used to perform operations on employees.

Possible actions may include:

| Action | Purpose |
|---|---|
| Create Employee | Add new employee |
| Edit Employee | Update employee details |
| Employee Transfer | Move employee between branches |
| Assign Contract | Connect employee to contract |
| Export | Export employee records |
| Delete/Deactivate | Disable employee profile |

The Employee List module is the central employee management screen of the HRM system.

It helps users:
- Manage employees
- Search records
